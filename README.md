## Build Process

### Dependencies

- Yarn

### Getting Started

1. Clone or download this repository.
   ```sh
   git clone https://github.com/gmaroko/otg-web.git
   cd otg-web
   ```
2. Install build dependencies in the project directory.
   ```sh
   yarn install
   ```

3. Run the web client with webpack for local development.
   ```sh
   yarn serve
   ```
