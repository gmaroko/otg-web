## Hack Process
1. Clone repo
   ```sh
   git clone https://github.com/gmaroko/otg-web.git
   cd otg-web
   ```
2. Install dependencies - using yarn (required)
   ```sh
   yarn install
   ```
3. Deploy to web client with webpack.
   ```sh
   yarn serve
   ```
